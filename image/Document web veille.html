LES TECHNOLOGIES FRONT END WEB ET MOBILE

QUEST CE QUE FRONT END

La notion de front end fait référence à l'ensemble des éléments visibles et accessibles directement d'un site web ou mobile; il est l'opposer du back end(partie invisible ). pour la réalisation de ce ensembles d'éléments visibles il faut une parfaite maitrise des langages de programmation et l'utilisation de framework.

QUEST CE QU'UN FRAMEWORK

Un framework est un ensemble coherent de composant logiciel structurels qui sert à creer les fonctions ainsi que les architectures.Cest une boite à outils pour un developpeur.
