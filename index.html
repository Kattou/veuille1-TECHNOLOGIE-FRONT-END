LES TECHNOLOGIES FRONT END WEB ET MOBILE

QUEST CE QUE FRONT END

La notion de front end fait référence à l'ensemble des éléments visibles et accessibles directement d'un site web ou mobile; il est l'opposer du back end(partie invisible ). pour la réalisation de ce ensembles d'éléments visibles il faut une parfaite maitrise des langages de programmation et ainsi devenir un concepteur-developpeur . De ce fait l'utilisation d'un framework est indispensable pour creer un logiciel ou une application.

QUEST CE QU'UN FRAMEWORK

Un framework est un ensemble coherent de composant logiciel structurels qui sert à creer les fonctions ainsi que les architectures.Cest une boite à outils pour un developpeur.les framework sont utilise pour facilité le developpement d'un site ou d'une application. Il faut l'utilise pour : son eficacité le framework limite la présence de code repete et le risque de faire des erreurs.

la securite

le framework est rapides deans la detection des potentielles failles de securite et leur correction sont rapides

l'evolution constante

Chaque framework dispose d'une communaute importante ce qui facilite la resolution de differents problemes

le cout

les frameworks les plus populaires sont gratuits.

QUELQUES FRAMEWORKS

Il existe plusieurs types de framework:ceux dits "logiciels",qui visent à developper en incluant notamment des bibliotheques de code,des programmes des compilateurs ou encore des API. A chaque langage de programmation son framework.Ainsi nous vous presenterons quelques frameworks.

Angular

cree par Google en 2009,Angular JS est un framework JavaScript open-source qui prend en charge l'animation,le routage,l'injection de dependance ou encore de data binding.il est tres populaire et dispose donc d'une communaute active aupres de laquelle vous pourrez facilement trouver de l'aide.

React JS

Il est certainement l'un des frameworks front-end les plus utilisés par les développeurs web. Grâce à React, il est facile de créer des interfaces utilisateurs interactives. Définissez des vues simples pour chaque état de votre application, et lorsque vos données changeront, React mettra à jour, de façon optimale, juste les composants qui en auront besoin.Il est aussi utilise en JavaScript

Django

Il a pour but de rendre le développement web 2.0 simple et rapide. Pour cette raison, le projet a pour slogan « Le framework pour les perfectionnistes avec des deadlines » Django est utilise avec Python.

Somfony 2

Il est assez facile de l’utiliser car sa communauté et très grande et fort active.Ce framework utilisé pour le langage de programmation PHP.

DEFINITION D'UN LANGAGE DE PROGRAMMATION

Un langage de programmation est une notion conventionnelle destinee à formuler des algorithmes et produire des programmes informatique qui les appliquent.

QUELQUES LANGUAGES DE PROGRAMMATION

Nous avons plusieurs types de langage ;de ce fait nous en citerons quelque un:

HTML et CSS:

l'HTML sert a definir les differents elements d'une page ,a leur donner du sens. Le CSS,lui va servir à mettre en forme les differents contenus definis par le HTML en leur appliquant des styles.

JAVA SCRIPT:

Cest un langage de programmation de scripts principalement employe dans les pages web interactives et a ce titre est une partie essentielle des applications web.il est au coeur des langages pour les developpeur web.

PHP:

Hypertext PREPROCESSOR plus connus sous son sigle PHP,cest un langage de programmation libre utilise pour produire des pages web dynamique via un serveur http

JAVA:

C'est un langage de programmation orienté objet cree par JAMES GOSLING et PATRICK NAUGHTON. Le langage Java sert à créer des logiciels dans des environnements très divers : applications sur client lourd (JFC); applications Web, côté serveur (servlets, JSP, Struts, JSF).
